Welcome to PyPolyhedralCubature's documentation!
================================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

This package allows to evaluate the integral of a multivariate function over 
a convex polytope, given by the linear combinations of the variables defining 
the bounds of the integrals.

See the `README <https://github.com/stla/PyPolyhedralCubature#readme>`_ file 
for usage examples.
